# Tahu Programming Language

## Quick Examples
### Hello World
```tahu
fn main() {
    println("Hello, World!")
}
```

### Variables and Types
```tahu
fn main() {
    // Immutable variable
    val name: string = "Tahu"

    // Mutable variable
    var age: int = 5
    age = 6 // Reassigning a mutable variable

    var port: int = 8080
    var is_running: bool = true
    
    // Type inference
    var message = "Server running on port ${port}"
    print(message)
}
```
